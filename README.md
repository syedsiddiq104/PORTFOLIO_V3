# Personal Portfolio

This is my personal portfolio website built to showcase my projects, skills, and work in frontend development. The goal was to create a clean, responsive, and visually engaging interface that reflects both technical ability and design thinking.

## Live Website

https://syedsiddiq104.github.io/PORTFOLIO_V3/

## Repository

https://github.com/syedsiddiq104/PORTFOLIO_V3

## Features

* Responsive layout for mobile, tablet, and desktop
* Animated sections and smooth transitions
* Project showcase with live and GitHub links
* Contact section with working form integration
* Modern UI with structured content flow

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Approach

The portfolio was designed with a focus on clarity and presentation. Each section is structured to guide the user through an overview, skills, and projects without unnecessary complexity. Animations and visual elements were added to enhance the experience while keeping performance in mind.
